Space Interior Mumbai offers bespoke luxury interiors, residential design, commercial spaces, and custom furniture. Transforming Spaces, Elevating Lifestyles.

## Run Locally

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Set the `GEMINI_API_KEY`:**
    Create a `.env` file (or update `.env.local`) and set:
    `GEMINI_API_KEY=your_actual_api_key_here`
3.  **Run the app:**
    ```bash
    npm run dev
    ```

## Technology Stack

-   **Frontend:** React, Vite
-   **Styling:** TailwindCSS
-   **AI Integration:** Google Gemini API
-   **Animations:** Framer Motion / Motion
